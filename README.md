## This is a budget breeze web and app for budget estimation.

** Key Features ** 
1. Email Password Login and Registration.
2. Google Auth
3. Logout
4. Refresh the page data.
5. Input Taken
6. Data Save and Read from the Firebase.
7. Realtime Data changing.
8. Full Responsible.
